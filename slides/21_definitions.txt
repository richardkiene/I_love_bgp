% Important definitions

* AS: Autonomous System controlled by a single organization

* ORIGIN: BGP speaker which originates routing information

* AS_PATH: AS_SET or AS_SEQUENCE defining the path which routing information was
  learned

* NEXT_HOP: IP address of the router to use when sending traffic

* MULTI_EXIT_DISC: Metric used to break a tie when all neighbors appear equal

* LOCAL_PREF: Used in iBGP and eBGP confederations to express a prefered
  external route.

* iBGP: BGP connection between internal (same AS) peers

* eBGP: BGP connection between external (different AS) peers
